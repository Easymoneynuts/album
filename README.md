# PANINI MUNDIAL 2022

Proyecto de galería de jugadores hecho con React. Muestra 5 jugadores de Inglaterra con sus fotos, posiciones, club actual y estadísticas de la selección.

## Qué es esto?

Es una página web sencilla donde salida un listado de jugadores y al hacer clic en uno ves toda la info. Nada muy complicado, básicamente un proyecto para practicar React.

## Funciones

- Galería con fotos
- Información de cada jugador:
  - Club donde juegan ahora
  - Cuántos partidos internacionales tienen
  - Goles metidos
  - Datos personales

## Cómo correr el proyecto

```bash
npm install
npm run dev
```

Se abre en http://localhost:5173/

## Si querés cambiar algo

Los datos de los jugadores están en `src/data/players.js`. Los estilos están en `src/styles/App.css`.

## Stack

- React
- Vite
- CSS3
